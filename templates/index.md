# Templates

Ready-to-use structural templates for the most common documentation types. Copy the relevant template, replace all placeholder text in `[brackets]`, and remove sections that don't apply.

---

| Template | Use when |
|----------|----------|
| [API Reference](api-reference-template.md) | Documenting an API endpoint, resource, or parameter set |
| [How-to Guide](how-to-guide-template.md) | Writing a task-based guide with numbered steps |
| [Concept Explainer](concept-explainer-template.md) | Explaining a concept, architecture, or mental model |
| [Release Notes](release-notes-template.md) | Publishing a versioned changelog or release summary |
